# Getting started

## Usage

Convert 8-byte hexadecimal string to 64-bit integer string

```js
const hexTo64Int = require('hex-to-64int');
hexTo64Int('100000a218da0085');
//=> 1152922200808489093
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
